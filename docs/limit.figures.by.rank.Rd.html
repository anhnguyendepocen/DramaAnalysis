limit.figures.by.rankR Documentation

This method removes the spoken tokens of all but the most frequent n figures

Description

This method removes the spoken tokens of all but the most frequent n figures

Usage

limit.figures.by.rank(t, maxRank = 10)

Arguments

t

The text, a data frame listing each token for each figure

maxTokens

Up to maxTokens figures remain in the data set

Examples

t <- load.text(c("rksp.0"), tokens=TRUE)
t <- limit.figures.by.rank(t)
