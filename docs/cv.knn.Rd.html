cv.knnR Documentation

Classification using k nearest neighbor Evaluation is done using cross validation.

Description

Classification using k nearest neighbor Evaluation is done using cross validation.

Usage

cv.knn(ft, labels, k = 5, num.folds = 10)

Arguments

ft

A frequency table. Words/lemmas in columns, texts/figures in rows

k

The number of neighbors for kNN

num.folds

The number of folds for CV

gold

A list of gold labels in the same order as in the frequency table

Examples

baseurl <- "http://zwergdrossel.ims.uni-stuttgart.de:8080/"
url.tragedies <- paste(baseurl, "drama.web/set/trag%C3%B6die", sep="")
url.comedies <- paste(baseurl, "drama.web/set/kom%C3%B6die", sep="")
meta.comedies <- read.csv(url.comedies)
meta.tragedies <- read.csv(url.tragedies)
text.all.comedies <- load.text(meta.comedies$id, tokens=TRUE)
text.all.tragedies <- load.text(meta.tragedies$id, tokens=TRUE)
