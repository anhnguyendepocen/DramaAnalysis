make_figure_statisticsR Documentation

This function extracts figure statistics from a drama text table.

Description

This function extracts figure statistics from a drama text table.

Usage

make_figure_statistics(t, names = FALSE, normalize = FALSE)

Arguments

t

The drama text

names

If set to true, the table will contains figure names instead of ids

normalize

Normalising the individual columns

Value

A data frame with the following columns and one row for each figure: tokens: The number of tokens spoken by that figure types : The number of different tokens (= types) spoken by each figure utterances: The number of utterances utterance_length_mean: The mean length of utterances utterance_length_sd: The standard deviation in utterance length

Examples

data(rksp.0)
stat <- make_figure_statistics(rksp.0, names = FALSE)
