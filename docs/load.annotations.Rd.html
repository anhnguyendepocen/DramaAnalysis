Description

Helper method to load covered annotations.

Usage

load.annotations(ids, type = "de.unistuttgart.ims.drama.api.Utterance",
  coveredType = "de.tudarmstadt.ukp.dkpro.core.api.segmentation.type.Token")

Arguments

ids

A vector or list of drama ids

type

The annotation type to load

coveredType

The annotation type of covered annotations we want to load

Examples

load.annotations(c("rksp.0"))
