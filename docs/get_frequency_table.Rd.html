get_frequency_tableR Documentation

Returns frequency tables for each figure in all dramas in t

Description

Returns frequency tables for each figure in all dramas in t

Usage

get_frequency_table(t, relative = FALSE, names = FALSE,
  accepted.pos = c())

Arguments

t

A data frame representing (one or more) text

relative

Whether to calculate relative frequencies

accepted.pos

A vector containing pos tags that should be included. If the vector is empty, includes everything.

If

set to true, the resulting table will include figure names instead of figure ids

Examples

data(rksp.0)
freq_table <- get_frequency_table(rksp.0)
