Description

This method retrieves word lists from the given URL (e.g., github) and counts the number of occurrences of the words in the dictionaries

Usage

dictionary.statistics(t, fieldnames = c(), normalize = FALSE,
  names = FALSE, boost = 100,
  baseurl = "https://raw.githubusercontent.com/quadrama/metadata/master/fields/")

Arguments

t

A text

fieldnames

A list of names for the dictionaries. It is expected that files with that name can be found below the URL.

normalize

Whether to normalize by figure speech length

names

Whether the resulting table contains figure ids or names

boost

A scaling factor to generate nicer values

baseurl

The url delivering the dictionairies
