utterance.statisticsR Documentation

This method calculates the length of each utterance, organised by figure and drama.

Description

This method calculates the length of each utterance, organised by figure and drama.

Usage

utterance.statistics(t, num.figures = 10)

Arguments

t

The dramatic text(s)

num.figures

The maximal number of figures per drama to include. Default: 10. Set to FALSE to include all figures.

Examples

data(rksp.0)
num_figures <- 5
ustat <- utterance.statistics(rksp.0, num.figures = num_figures)
boxplot(ustat$utterance_length ~ ustat$figure,col=qd.colors[1:num_figures], las=2,frame=F)
