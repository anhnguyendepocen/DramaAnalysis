This function generates a table to be used by stylo

Description

This function generates a table to be used by stylo

Usage

frequency.table(t)

Arguments

t

The text table, potentially covering multiple texts

accepted.pos

A list of accepted pos tags

names

Whether to use figure names or ids

column

The column name we should use (should be either Token.surface or Token.lemma)

by.figure

Wether the count is by figure or by text

Examples

data(rksp.0)
st <- frequency.table(rksp.0)
stylo(gui=FALSE, st)
